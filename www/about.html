📖 About This Project

🎯 Project Objectives

Develop a multi-threaded web server in Python that processes HTTP requests from browsers or client programs using raw socket programming.

⚙️ Technical Details

  • 🐍 Language: Python 3
  • 🔌 Socket programming – no high-level HTTP libraries
  • 🧵 Threading: one thread per incoming request
  • 📁 Document root: ./www/ folder
  • 📝 Logging: records client IP, time, requested file, and status code

📊 HTTP Status Codes Supported

  • ✅ 200 OK – successful request
  • ⚠️ 400 Bad Request – malformed HTTP request
  • 🔒 403 Forbidden – access denied
  • ❓ 404 Not Found – file does not exist
  • 🔄 304 Not Modified – file unchanged (If-Modified-Since)
